## Full Project Recipe

* React JS
* React Router DOM
* Axios
* Styled Components

## Setup

```bash
$ git clone kdjbnv.git
$ cd social-network-fe
$ npm install
```
Open [LocalHost](http://localhost:3000)

## Start server


Make sure to start backend server first. Only then, start the frontend server.
```bash
$ npm start
```
